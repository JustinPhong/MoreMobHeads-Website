What is More Mob Heads - Extension

More Mob Heads Extension is a lightweight Minecraft Paper plugin designed to work in tandem with the More Mob Heads datapack. Its purpose is to extend and refine the head-dropping mechanics in Minecraft while preserving the original vanilla behavior.

In vanilla Minecraft, certain mob heads like skeleton, zombie, and creeper heads are only obtainable when a mob is killed by a charged creeper.

Features

  • Vanilla-accurate mob head drops: heads only drop when killed by charged creepers
  • Supports all player head drops with configurable chance
  • Integrates seamlessly with the More Mob Heads datapack loot tables
  • Lightweight and server-friendly, no heavy dependencies
  • Easy installation: just place plugin in plugins and datapack in world/datapacks

How It Works

When a mob or player dies, the plugin checks if the last damage was caused by a charged creeper. If so, it triggers the More Mob Heads datapack loot table and drops the corresponding head. Otherwise, no head is dropped.
