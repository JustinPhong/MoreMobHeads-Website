Installation

1. Download the MoreMobHeads-Extension-datapack-X.X.zip and the MoreMobHeads-Extension-X.X.jar .
2. Place the datapack in server/world/datapacks.
3. Place the plugin jar in server/plugins.
4. Restart your server or use /reload.

Verify installation: /plugins to check plugin, /datapack list to check datapack.

Requirements

  • PaperMC 1.21+ (supports minor versions 1.21.x)
  • Java 17 or newer
